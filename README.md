# 🧠 Quiz Guess Game

A simple and interactive quiz application built with React.  
Users answer multiple-choice questions, get instant feedback, and see their final score at the end.

---

## 🚀 Live Demo
https://edogistar.github.io/quiz-guess-game/

---

## 🛠️ Built With

- React (Vite)
- JavaScript (ES6+)
- HTML5
- Tailwind CSS

---

## 🎯 Features

- Multiple-choice quiz questions
- Option selection with highlight effect
- Score tracking system
- Next question navigation
- Final result screen
- Restart quiz functionality
- Responsive and clean UI

---

## 🧠 How It Works

- Questions are stored in a local data file
- React state manages:
  - Current question index
  - Selected option
  - Score
  - Quiz completion state
- Score updates when user proceeds to next question
- Final screen shows total score and restart option
